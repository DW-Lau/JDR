JDR - Application Web, lancer de dé.

Fonctionnement Basic Fonctionnement CheatMod
  • Ajoutez un "bonus" ou un "malus" dans les options, si nécéssaire.
  • Puis sélectionnez un dé.
  • Appuyez sur le bouton de lancer et obtenez votre résultat, après calcul des options.
 Envie de tricher ? ;)
  • Sélectionner le mode "CheatMod" pour activer les réussites ou échecs critiques.
  • Puis sélectionnez un dé.
  • Appuyez sur le bouton de lancer. Ici les autres options sont désactivées.
